# Draw Tools

The Draw category contains tools for creating procedural elements, gradients, noise patterns, grain, lens flares, and other drawn effects in Nuke.

## Submenus

- [Expression Nodes AG](expression-ag/index.md) - Expression-based tools by Andrea Geremia
- [Hagbarth Tools](hagbarth/index.md) - Creative processing tools by Mads Hagbarth Damsbo

## Tools

| Tool | Author | Description |
|------|--------|-------------|
| [GradMagic](grad-magic.md) | TL | Live sampling 4 point gradient tool with ability to bake colors |
| [NoiseAdvanced](noise-advanced.md) | TL | Noise with user friendly animation sliders and overscan |
| [RadialAdvanced](radial-advanced.md) | TL | Radial tool that creates a circle and ramped falloff "ring" effect |
| [WaterLens](water-lens.md) | MJT | Create water droplet on lens |
| [VoronoiGradient](voronoi-gradient.md) | NKPD | 2D Gradients using Natural Neighbor Interpolation |
| [CellNoise](cell-noise.md) | NKPD | 6 cellular noise types with standard nuke noise controls |
| [LineTool](line-tool.md) | NKPD | Line drawing gizmo for drawing lines on input |
| [PlotScanline](plot-scanline.md) | NKPD | Slice and plot scanlines in Nuke |
| [SliceTool](slice-tool.md) | FR | Analyze an arbitrary slice of an image |
| [PerspectiveGuide](perspective-guide.md) | NKPD | Simple perspective guide for setting horizon and perspective lines |
| [DasGrain](das-grain.md) | FH | Semiautomatic regraining tool |
| [LumaGrain](luma-grain.md) | NKPD | Added functionality to Nuke's default grain node |
| [GrainAdvanced](grain-advanced.md) | SPIN | Adds synthetic grain resembling HD Alexa plate grain |
| [X_Tesla](x-tesla.md) | XM | Create lightning and electricity effects |
| [FlareSuperStar](flare-super-star.md) | NKPD | Create star flares with separate controls over rays, glare and glow |
| [AutoFlare](auto-flare.md) | NKPD | Automatic lens flare filter based on image content |
