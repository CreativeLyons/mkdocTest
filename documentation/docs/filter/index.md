# Filter

The Filter category contains tools for glows, blurs, edge treatment, distortions, chromatic aberration, lightwraps, and various image filtering effects.

## Submenus

- [Glows](glows/index.md) - Exponential glow effects
- [Blur](blur/index.md) - Advanced blur tools
- [Edges](edges/index.md) - Edge detection, extension, and treatment
- [Distortions](distort/index.md) - Glass, heat wave, and distortion effects
- [X_Tools XM](x-tools/index.md) - Xavier Martin's GPU-accelerated tools
