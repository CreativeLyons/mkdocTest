# Blur

Advanced blur tools beyond Nuke's standard blur nodes.

## Tools

| Tool | Author | Description |
|------|--------|-------------|
| [ExponBlurSimple](expon-blur-simple.md) | Tony Lyons | Exponential blur with iterating steps |
| [Directional Blur](directional-blur.md) | Tony Lyons | Blur at specific angle with perpendicular option |
| [IBlur](iblur.md) | Moritz Eiche | Smoothly ramped blur based on matte |
| [WaveletBlur](wavelet-blur.md) | Mads Hagbarth Damsbo | Pick specific frequency range to blur |
| [FractalBlur](fractal-blur.md) | Richard Frazer | Blur with noise for textured edges |
